# 💍 Digital Wedding Gallery

---

### 📖 About This Project

This repository contains the front-end code for a digital wedding photo gallery. 

Building a photo-heavy web application presents unique front-end challenges, specifically regarding load times, performance optimization, and responsive design. I built this gallery to provide a seamless, elegant user experience for viewing high-resolution images across all device sizes, from desktop monitors to mobile phones.

This project demonstrates my ability to craft polished user interfaces and manage asset-heavy web applications efficiently.

### ✨ Key Features & Performance Optimizations

* **Responsive Masonry Layout:** Dynamically arranges images of varying aspect ratios into a clean, interlocking grid without awkward cropping.
* **Lazy Loading:** Images are only loaded as they enter the browser's viewport, drastically reducing initial page load times and saving bandwidth.
* **Lightbox / Modal Viewing:** Users can click on any image to view it in a full-screen, immersive modal with smooth transitions.
* **Optimized Asset Delivery:** Configured to handle compressed, next-gen image formats (like WebP) to ensure high performance.

### 💻 Local Development

To run this gallery locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/shamooo123/wedding-gallery.git](https://github.com/shamooo123/wedding-gallery.git)

``

2. **Navigate into the directory:**
```bash
cd wedding-gallery

```


3. **Install dependencies:**
```bash
npm install

```


4. **Start the development server:**
```bash
npm run dev

```
